# FastAPI Backend for ScribbleArchitect

This is the FastAPI backend for the ScribbleArchitect web application.

## Setup

```bash
cd backend
pip install -r requirements.txt
```

## Running

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 5050
```

## API Documentation

Once running, visit:
- Swagger UI: http://localhost:5050/docs
- ReDoc: http://localhost:5050/redoc
